To use the Dummy [*] example, please:

-	copy the cfg_application_*.json file in the same folder as the cControl.m file and make the cCompute*.m available for the interpreter.
-	type "cControl" in the Matlab command window
-	a list dialog should now appear with "*" as the only choice. -> choose it!
-	a load dialog will appear. Please select the "DataFolder" folder
-	a question dialog will appear. You can freely choose to start a new session or use the existing one the program found in the folder.
